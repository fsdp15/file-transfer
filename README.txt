This program only works if you connect 2 computers using a crossover cable. It is supposed to simulate the link layer, so it doesn't use
the internet or a local network.

Available options as a client:

lsl (local ls)
lsr (remote ls)
cdl (local cd)
cdr (remote cd)
get "file"
put "file"
